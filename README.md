# 📅 Interactive Wall Calendar

A fully responsive and interactive wall-style calendar built using **HTML, CSS, and JavaScript**.
This project focuses on strong frontend fundamentals while delivering a polished and user-friendly experience.

## ✨ Features

### 📆 Calendar UI

* Wall calendar aesthetic with **image-based background**
* Monthly navigation (Previous / Next)
* Fully responsive (Desktop + Mobile)

### 🎯 Date Interaction

* Click to **select a date**
* **Date range selection** (start → end)
* Highlighted range for better visibility

### 📝 Notes System

* Add notes for **individual dates**
* Notes saved using **localStorage**
* Persistent even after page reload

### 🎨 UI Enhancements

* Sunday hover → 🔴 Red
* Saturday hover → 🟡 Yellow
* Other days hover → 🔵 Blue
* Smooth hover animations

### 🌙 Dark Mode

* Toggle between **Light / Dark mode**
* Theme preference saved automatically

---

## 📱 Responsive Design

* Desktop: Side-by-side layout (Calendar + Notes)
* Mobile: Stacked layout for better usability
* Optimized for touch interactions

---

## 🛠 Tech Stack

* **HTML5**
